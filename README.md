OptiCrop – Smart Agricultural Production Optimization Engine
------
Project Overview
------
OptiCrop is an AI-powered crop recommendation system that predicts the most suitable crop based on soil nutrients and environmental conditions. The application helps users make better agricultural decisions using Machine Learning.
Repository Structure
------
This repository follows the AI-ML-and-GEN-AI Track project structure.
1. Brainstorming & Ideation
Contains brainstorming activities, idea generation, problem identification, empathy map, and project proposal.
2. Requirement Analysis
Contains the problem statement, functional requirements, non-functional requirements, hardware and software requirements, and feasibility analysis.
3. Project Design Phase
Contains the system architecture, Data Flow Diagram (DFD), use case diagram, flowchart, and UI design.
4. Project Planning Phase
Contains the project workflow, milestones, timeline, task allocation, and planning documents.
5. Project Development Phase
Contains the complete source code, trained ML model, dataset, Flask application, HTML templates, CSS files, and required libraries.
6. Project Testing
Contains test cases, test results, bug reports, validation documents, and screenshots.
7. Project Documentation
Contains the project report, user manual, installation guide, and supporting documents.
8. Project Demonstration
Contains the project presentation, demonstration video, screenshots, and demo materials.

Installation
------
1.	Clone this repository.
2.	Install the required Python packages:
pip install -r requirements.txt
3.	Run the application:
python app.py
Project Workflow
------
•	Enter soil and environmental parameters.
•	Process the input using the trained machine learning model.
•	Predict the most suitable crop.
•	Display the recommended crop to the user.
